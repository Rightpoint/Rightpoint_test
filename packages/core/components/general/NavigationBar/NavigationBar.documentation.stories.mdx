## What it does

The navigation bar is a component that sticks to the top when the user scrolls past it.

## Where it scrolls to

When an a link is clicked, the user will be scrolled to the first element that matches the link title.

For example, if the link title is "Industries", the browser will scroll the user to the first element that contains the text "Industries".
